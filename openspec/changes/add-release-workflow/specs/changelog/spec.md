## ADDED Requirements

### Requirement: Changelog follows keepachangelog format
The project SHALL have a `CHANGELOG.md` at the repository root following the keepachangelog.com format with an `## [Unreleased]` section at the top.

#### Scenario: Initial changelog
- **WHEN** the changelog is created
- **THEN** it SHALL contain a header, an `## [Unreleased]` section, and a `## [0.1.0]` entry for the initial release
